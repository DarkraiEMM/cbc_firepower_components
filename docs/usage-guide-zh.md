# CBC：火力组件使用说明

CBC：火力组件是 Create Big Cannons 的附属模组，主要补充占地更小的炮座，以及更适合自动化的供弹、装填组件。它不是替代 CBC 原有炮座，而是给紧凑炮塔、车辆炮塔、舰载炮位和自动供弹结构提供一些更方便的选择。

## 版本和依赖

目前有两条发布线：

- Minecraft 1.21.1 / NeoForge 21.1.x
- Minecraft 1.20.1 / Forge 47.x

需要安装：

- Create
- Create Big Cannons
- Ritchie's Projectile Library

## 新增内容

- 紧凑火炮底座：一格大小的通用紧凑炮座，可用于 CBC 大炮、迫击炮、机炮和手摇机炮。
- 紧凑机关炮炮台：专门用于机炮的轻型炮座。
- 机关炮供弹器：把散装机炮弹药送入相邻、已经装配好的紧凑炮座。
- 火炮弹匣装填器：三发外置弹匣，用于给已装配的大炮炮座自动送入弹头和有药大型弹药。

## 炮座通用控制

两个紧凑炮座都使用接近 CBC 原版炮座的控制逻辑，但面更少，所以摆放方向很重要。

- 正面：红石开火控制。红石信号强度会影响射速。
- 背面：红石装配/拆卸控制。
- 上方或下方传动杆接口：控制水平旋转。
- 左右侧传动杆接口：控制俯仰角。
- 扳手点击正面或背面：交换开火面和装配面。
- 扳手点击侧面：调整炮座水平朝向。
- 扳手点击上面或下面：通用紧凑火炮底座会旋转水平朝向；紧凑机关炮炮台会切换机炮安装面。

装配成功后，物品输入会转发给已经安装的火炮。漏斗、漏槽、机械臂等常规物品自动化可以直接对接炮座，或对接本模组的供弹、装填组件。

## 紧凑火炮底座

![紧凑火炮底座配方](images/recipe-compact-cannon-mount.png)

![紧凑火炮底座安装侧](images/placement-compact-cannon-mount.png)

紧凑火炮底座是通用炮座。

可以安装：

- CBC 大炮
- CBC 迫击炮
- CBC 机炮
- CBC 手摇机炮

放置和安装：

1. 放置紧凑火炮底座。
2. 将火炮结构放在炮座左侧伸出的传动杆一侧。
3. 给背面输入红石信号，炮座会尝试装配相邻的火炮。
4. 给正面输入红石信号开火。

面和输入：

- 左侧伸出的传动杆一侧：放置要装配的火炮。
- 正面：只用于红石开火控制。
- 背面：只用于红石装配/拆卸控制。
- 上方/下方轴口：接入旋转动力，控制水平旋转。
- 左右轴口：接入旋转动力，控制俯仰角。
- 方块物品输入：装配后会转发给已安装火炮，用于普通装填或机炮供弹。

方向调整：

- 用扳手点击正面或背面，可以交换开火面和装配面。
- 用扳手点击侧面，可以改变炮座朝向。
- 用扳手点击上面或下面，会让炮座水平旋转。
- 如果想改变火炮相对炮座的上下安装状态，建议拆下后重新放置炮座。

## 紧凑机关炮炮台

![紧凑机关炮炮台配方](images/recipe-compact-autocannon-mount.png)

紧凑机关炮炮台是机炮专用炮座，不能安装大炮或迫击炮。

可以安装：

- CBC 机炮

放置和安装：

1. 放置紧凑机关炮炮台。
2. 在炮台当前的安装面放置机炮。
3. 给背面输入红石信号，炮台会尝试装配机炮。
4. 给正面输入红石信号开火。

面和输入：

- 机炮安装面：放置机炮。
- 正面：红石开火控制。
- 背面：红石装配/拆卸控制。
- 上方/下方轴口：接入旋转动力，控制水平旋转。
- 左右轴口：接入旋转动力，控制俯仰角。
- 方块物品输入：装配后会供给已安装的机炮。

方向调整：

- 用扳手点击正面或背面，可以交换开火面和装配面。
- 用扳手点击侧面，可以改变炮台朝向。
- 用扳手点击上面或下面，可以切换机炮安装面。

紧凑机关炮炮台推荐配合机关炮供弹器使用。

## 机关炮供弹器

![机关炮供弹器配方](images/recipe-autocannon-ammo-feed.png)

机关炮供弹器用于把散装机炮弹药自动送入相邻的紧凑机关炮炮台。只要放置在炮台旁边，供弹器的弹链会自动朝向并插入炮台，这样就可以使用漏斗、机械臂之类的装置输入弹药。

可以输入：

- 散装机炮弹药

不能输入：

- 整个机炮弹药箱
- 大炮弹头
- 大型弹药
- 其他普通物品

放置和使用：

1. 将机关炮供弹器放在紧凑机关炮炮台旁边。
2. 用漏斗、漏槽、机械臂等方式向供弹器输入散装机炮弹药。
3. 供弹器会自动转向相邻炮台，并持续尝试把弹药送入已经装配好的机炮。

面和输入：

- 任意面都可以接受散装机炮弹药输入。
- 供弹器没有固定输出面；它的弹链会朝向相邻的紧凑机关炮炮台。
- 如果相邻炮台未装配，或机炮输入缓存已满，弹药会留在供弹器中。

注意：

- 供弹器只处理散装机炮弹药，不处理整箱机炮弹药。
- 供弹器需要贴着炮台放置；隔一格或隔着其他方块都不会送弹。
- 已经装入弹药的机炮弹药箱可以直接输入到炮座本身，不需要经过供弹器。

## 火炮弹匣装填器

![火炮弹匣装填器配方](images/recipe-cannon-magazine-loader.png)

火炮弹匣装填器是三发外置弹匣，用于给已装配的大炮炮座供弹，让火炮实现更快的连续射击。它不是给机炮用的，贴着炮座放置即可。

可以输入：

- CBC 大炮弹头
- 有药大型弹药

不能输入：

- 散装机炮弹药
- 机炮弹药箱
- 空大型弹壳作为待发药筒
- 普通物品

最重要的槽位规则：

- 上半区三格：放弹头。
- 下半区三格：放有药大型弹药。
- 同一列是一组弹药：上面弹头，下面对应药筒。
- 自动装填时，会先送入弹头，再送入同列的有药大型弹药。

手动使用：

1. 手持 CBC 大炮弹头右键装填器，会放入上半区弹头槽。
2. 手持有药大型弹药右键装填器，会放入下半区药筒槽。
3. 空手右键对应位置，可以取出该位置的物品。

自动化输入：

- 自动输入弹头时，会优先填入上半区空槽。
- 自动输入有药大型弹药时，只会放到已经有弹头的同列下方。
- 通过自动化开始填充后，装填器会等三组弹药全部齐备，再开始送弹循环。
- 手动装填和取出始终可用，不受自动化等待逻辑限制。

与炮座的关系：

- 装填器需要放在已装配的大炮炮座旁边。
- 它可以向相邻的紧凑火炮底座或 CBC 大炮炮座送弹。
- 它会按“弹头 -> 对应有药大型弹药”的顺序向炮内装填。

空弹壳输出：

- 装填器会尝试从相邻的大炮中取出已经击发后的空大型弹壳。
- 空大型弹壳会进入装填器内部空槽。
- 侧面、背面和底面可以被漏斗、漏槽等自动化抽出空大型弹壳。
- 正面和顶面更适合作为输入面，不作为空弹壳输出面。
- 自动化不会从输出面抽出未击发的弹头或有药大型弹药。

## 常见问题

炮座没有装配火炮：

- 检查火炮是否放在正确的安装面。
- 检查背面是否有红石信号。
- 检查火炮结构是否本身可以被 CBC 正常装配。

炮座不能转动：

- 检查水平旋转和俯仰是否接到了正确的传动杆接口。
- 检查动力方向和转速。
- 检查整门炮是否因为方块碰撞或装配限制卡住。

机关炮供弹器不送弹：

- 确认输入的是散装机炮弹药。
- 确认相邻炮座已经装配，并且装的是机炮。
- 确认机炮的输入缓存没有满。

火炮弹匣装填器不开始自动送弹：

- 如果使用自动化输入，装填器会等三列全部填好后再开始循环。
- 确认每列都有弹头和对应的有药大型弹药。
- 确认装填器相邻的是已经装配好的大炮炮座。
